# BeogradskaHeraldika

BeogradskaHeraldika je web sajt posvećen heraldici grada Beograda i simbolici grbova njegovih opština. Sajt pruža slike i osnovne informacije o grbovima beogradskih opština, njihovim elementima i značenju kroz jednostavnu i preglednu web strukturu.

**Projekat je razvijen u saradnji sa kolegom Lukom Stefanovićem.**

## Technologies Used

- HTML
- CSS
- Apache Web Server

## Description

Projekat je realizovan kao statički web sajt korišćenjem HTML-a za strukturu stranica i CSS-a za stilizaciju i vizuelni izgled. Cilj sajta je da na jasan i pristupačan način predstavi heraldiku Beograda i simboliku grbova njegovih opština.
